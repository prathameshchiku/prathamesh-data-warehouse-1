**SQL Data Warehouse Project**

**📝 Project Overview**
This project demonstrates the end-to-end development of a modern SQL Data Warehouse from scratch. It follows industry best practices for data engineering, including architecture design, ETL processes, and data modeling.

**🏗️ Data Architecture**
Bronze Layer: Raw data ingestion from source systems (CRM & ERP).
Silver Layer: Data cleaning, transformation, and integration.
Gold Layer: Business-ready star schema model for analytics and reporting.

**📂 Repository Structure**
/data-sets: Contains raw source CSV files.
/scripts: SQL DDL and ETL stored procedures organized by layer (/bronze, /silver, /gold).
/docs: Project documentation, including data flow, architecture diagrams, and data dictionary.
/tests: SQL scripts for data quality and integrity validation.

**🛠️ Technologies Used**
SQL Server & SQL Server Management Studio (SSMS)
Git/GitHub for version control
Draw.io for architecture and lineage diagrams
Notion for project management
